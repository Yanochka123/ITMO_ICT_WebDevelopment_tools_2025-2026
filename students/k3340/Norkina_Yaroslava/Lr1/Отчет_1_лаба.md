Получить задачу по ID: GET /tasks/1

Создать задачу: POST /tasks с JSON-телом

Обновить задачу: PUT /tasks/1

Удалить задачу: DELETE /tasks/1

Получить задачи с приближающимся дедлайном: GET /tasks/upcoming_deadlines?hours=48

Получить статистику: GET /tasks/statistics

Добавить подзадачу: POST /tasks/1/subtasks

Добавить временную запись: POST /tasks/1/time_entries

Управление категориями: GET/POST/DELETE /categories

Управление тегами: GET/POST/DELETE /tags

Код полностью соответствует требованиям:

✅ Временная БД с 3 записями

✅ Главная таблица имеет одиночный вложенный объект (category)

✅ Главная таблица имеет список объектов (tags, subtasks, time_entries)

✅ Реализованы CRUD операции для главной таблицы

✅ Реализованы модели и API для вложенных объектов

✅ Использованы Pydantic модели

✅ Аннотированы эндпоинты
